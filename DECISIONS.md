# 주차가능성예측앱 결정 기록

**작성일:** 2026-04-28

## 확정한 추천안

- **MVP 지역:** 서울
- **임시 서비스명:** 주차될까
- **지도:** Kakao Maps 기준
- **앱 형태:** 웹/PWA 우선
- **1차 데이터 범위:** 서울 공영주차장 실시간 주차정보
- **예측 방식 v0:** 머신러닝이 아니라 설명 가능한 규칙 기반 점수

## 이유

### 서울 우선

- 공영주차장 수요가 크고 검증 가능한 사용 시나리오가 많다.
- 서울 열린데이터광장의 `GetParkingInfo` 샘플 API가 현재 정상 응답한다.
- 실시간 주차 대수, 전체 면수, 갱신 시각, 운영시간, 요금 등 MVP에 필요한 핵심 필드가 확인됐다.

### Kakao Maps 우선

- 한국 주소/장소 검색 UX가 좋다.
- 목적지 검색, 주소 좌표 변환, 지도 표시, 길찾기 연결을 한 흐름으로 만들기 쉽다.
- 단, API 키와 호출량 제한을 초기부터 분리 관리한다.

### PWA 우선

- 설치 없이 모바일에서 바로 검증 가능하다.
- 위치 기반 앱 특성상 네이티브 앱보다 MVP 배포 속도가 빠르다.
- 필요하면 추후 React Native/Capacitor로 확장한다.

## 보류한 것

- 민영주차장 제휴
- 예약/결제
- 푸시 알림
- ML 기반 예측
- 회원가입
