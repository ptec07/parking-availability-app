# 주차될까 MVP 제품 요구사항 문서

**작성일:** 2026-04-28  
**제품명:** 주차될까  
**형태:** 서울 공영주차장 주차 가능성 예측 PWA

---

## 1. 제품 목표

서울에서 목적지 주변 공영주차장의 현재 혼잡도와 도착 시점의 주차 가능성을 한눈에 보여준다.

사용자가 해결하려는 질문:

> “여기로 가면 주차할 수 있을까? 안 되면 어디로 가야 할까?”

---

## 2. MVP 범위

### 포함

- 서울 공영주차장 데이터
- 목적지 검색
- Kakao Maps 지도 표시
- 주차장 리스트 표시
- 현재 잔여면수 계산
- 가능성 점수 v0
- 도착 예정 시간 선택
- 길찾기 링크
- 대체 주차장 추천

### 제외

- 회원가입
- 예약/결제
- 전국 주차장
- 민영주차장 제휴
- 푸시 알림
- 머신러닝 예측
- 사용자 리뷰/커뮤니티

## 3. 주요 사용자 시나리오

### 시나리오 A: 지금 출발하는 사용자

1. 사용자가 `강남역`을 검색한다.
2. 지도에 주변 공영주차장이 표시된다.
3. 각 주차장 카드에 가능성 점수와 잔여면수가 표시된다.
4. 사용자는 가능성 높은 주차장을 선택한다.
5. Kakao 길찾기로 이동한다.

### 시나리오 B: 30분 뒤 도착 예정 사용자

1. 사용자가 목적지를 검색한다.
2. 도착 시간을 `30분 뒤`로 바꾼다.
3. 앱이 시간대 위험도를 반영한 가능성 점수를 다시 계산한다.
4. 앱은 현재는 여유 있어도 도착 시점에 위험한 주차장을 낮게 표시한다.

### 시나리오 C: 1순위 주차장이 위험한 경우

1. 사용자가 가장 가까운 주차장을 선택한다.
2. 앱이 “가능성 낮음”과 이유를 보여준다.
3. 앱이 도보 6분 거리의 대체 주차장을 추천한다.

## 4. 화면 요구사항

### 4.1 홈 화면

- 목적지 검색창
- 현재 위치 주변 보기 버튼
- 추천 문구: “목적지를 입력하면 주차 가능성을 예측해드려요”

### 4.2 지도/결과 화면

- Kakao Maps 지도
- 주차장 핀 색상
  - 초록: 가능성 높음
  - 노랑: 보통
  - 빨강: 낮음
  - 회색: 데이터 불충분
- 하단 주차장 카드 리스트
- 정렬 옵션
  - 가능성 높은 순
  - 가까운 순
  - 저렴한 순

### 4.3 주차장 카드

표시 항목:

- 주차장명
- 가능성 라벨과 점수
- 잔여면수 / 전체면수
- 도보 예상 시간
- 요금 요약
- 갱신 시각
- 추천 이유 1줄

예시:

```text
세종로 공영주차장
가능성 높음 · 82점
잔여 803 / 전체 1260면 · 3분 전 갱신
목적지까지 도보 5분
이유: 잔여면수가 많고 데이터가 최신입니다.
```

### 4.4 상세 화면

- 주차 가능성 점수
- 점수 산출 이유
- 운영시간
- 요금
- 전화번호
- 길찾기 버튼
- 대체 주차장 3개

## 5. 가능성 점수 v0

### 5.1 입력값

- `total_spaces`: 전체면수
- `occupied_spaces`: 현재 주차 차량 수
- `available_spaces`: 전체면수 - 현재 주차 차량 수
- `updated_at`: 갱신 시각
- `distance_m`: 목적지와 주차장 거리
- `arrival_time`: 도착 예정 시각
- `is_open`: 도착 시점 운영 여부

### 5.2 점수 규칙

```text
if not is_open:
  score = 0
else:
  score = availability_score + freshness_score + distance_score + time_score
```

배점:

- 잔여면수 비율: 60점
- 데이터 신선도: 15점
- 목적지 거리: 10점
- 시간대 위험도: 15점

### 5.3 라벨

- 75~100: 가능성 높음
- 50~74: 보통
- 25~49: 가능성 낮음
- 0~24: 어려움
- 데이터 부족: 확인 필요

## 7. 운영상 주의 문구

앱은 주차를 보장하지 않는다.

UI 문구:

> 주차 가능성은 공공데이터와 도착 예정 시간을 바탕으로 계산한 예측입니다. 실제 현장 상황과 다를 수 있습니다.

---

## 8. MVP 성공 기준

- 사용자가 목적지를 검색하고 3초 안에 주차장 후보를 볼 수 있다.
- 각 주차장 카드에서 왜 추천됐는지 이해할 수 있다.
- 실시간 데이터가 오래된 경우 앱이 이를 명확히 표시한다.
- 주차 가능성 점수가 없는 것보다 의사결정에 도움이 된다고 느껴진다.
